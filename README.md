## Описание

RSS агрегатор для чтения новостных лент. Позволяет добавлять RSS-потоки и просматривать посты в удобном интерфейсе.

## Технологии

- Vite
- Bootstrap 5
- JavaScript (ES Modules)
- Promises (без async/await)

## Установка

```bash
make install
make dev
```
